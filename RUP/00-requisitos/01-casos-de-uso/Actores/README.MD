# Actores del Sistema

Diferentes tipos de usuarios y agentes que interactúan con el sistema CGU.

---

## Actores Identificados

- **Administrador**: Gestión global del sistema y usuarios.
- **Profesor**: Gestión de asistencias y validación de dispensas.
- **Alumno**: Solicitud de dispensas y consulta de estado.
- **Secretaria**: Gestión administrativa de matrículas y listados.
- **Director de Grado**: Validación final de trámites académicos.
